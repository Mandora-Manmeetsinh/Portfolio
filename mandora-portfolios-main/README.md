## ğŸš€ Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mandora-Manmeetsinh/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
